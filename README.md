# Audio Synthesis Engine

## Brief Introduction
A C++ procedural audio synthesis engine that generates waveforms using custom oscillators and exports them to WAV files.
It was also built from scratch without using DSP libraries.

## Motivation
This project was created to understand how digital synthesizers work internally, including waveform generation, phase accumulation, and audio file encoding.

## Future Work
- More oscillators
- ADSR envelope generator
- Filters (low-pass, high-pass)
- Audio effects (reverb, delay, distortion)
